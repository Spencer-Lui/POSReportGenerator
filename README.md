# POS Report Generator

A desktop POS report generator written in Modern C++.

## Features

- SQLite Database
- Dear ImGui UI
- Repository Pattern
- Service Layer
- Layered Architecture

## 🛠 Tech Stack

- C++17
- Dear ImGui
- SQLite3
- Visual Studio 2026
